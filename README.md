# Computer Components Website

An educational website about computer hardware components with detailed information pages for each major part of a computer system.

## Pages

- **Home** (`index.html`) - Overview of all computer components and how they work together
- **CPU** (`cpu.html`) - Central Processing Unit information and specifications
- **GPU** (`gpu.html`) - Graphics Processing Unit details and comparison
- **RAM** (`ram.html`) - Random Access Memory types and capacity guidelines
- **Storage** (`storage.html`) - Storage devices (SSD, HDD) and interfaces
- **Motherboard** (`motherboard.html`) - Main circuit board information and form factors
- **Power Supply** (`psu.html`) - PSU ratings, efficiency, and connectors
- **Cooling** (`cooling.html`) - Air and liquid cooling solutions
- **Case** (`case.html`) - Computer case types and features

## Features

- Modern, responsive design with gradient color scheme
- Mobile-friendly navigation with hamburger menu
- Detailed information about each computer component
- Component comparisons and specifications
- Usage guidelines and recommendations
- Smooth animations and transitions
- Clean, educational layout

## Technologies Used

- HTML5 for semantic structure
- CSS3 with modern features (Grid, Flexbox, animations)
- Vanilla JavaScript for interactivity
- Responsive design for all screen sizes

## Getting Started

Simply open `index.html` in a web browser to start exploring the website. All pages are interconnected through the navigation menu.
