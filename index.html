How Components Work Together

1

Power On

The PSU supplies power to the motherboard and all components.

2

Boot Process

The CPU reads BIOS/UEFI from the motherboard and starts the operating system from storage.

3

Load Programs

Programs are loaded from storage into RAM for faster access by the CPU.

4

Execute Tasks

The CPU processes instructions while the GPU handles graphics and the cooling system manages temperature.
