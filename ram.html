Random Access Memory (RAM)

Your computer's short-term memory

What is RAM?

Random Access Memory (RAM) is a type of computer memory that can be read and changed in any order, typically used to store working data and machine code. It's volatile memory, meaning it loses its contents when the computer is turned off. RAM allows your computer to access data quickly, which is essential for smooth multitasking and overall system performance.

How RAM Works

RAM acts as a middle ground between your fast CPU and slower storage drives. When you open a program or file, it's loaded from storage into RAM so the CPU can access it much faster. The more RAM you have, the more programs and data you can work with simultaneously without slowing down your system.

Types of RAM

DDR4

Fourth generation DDR RAM. Most common in systems from 2016-2022. Good balance of performance and cost.

  • Speeds: 2133-3200 MHz
  • Voltage: 1.2V
  • Still widely used

DDR5

Latest generation DDR RAM. Higher bandwidth and efficiency. Standard for new systems.

  • Speeds: 4800-7200+ MHz
  • Voltage: 1.1V
  • Better for modern CPUs

LPDDR

Low-power DDR RAM used in laptops and mobile devices. Optimized for battery life.

  • Lower power consumption
  • Soldered to motherboard
  • Common in ultrabooks

Key RAM Specifications

Capacity

Amount of memory available, measured in GB. More capacity allows for more simultaneous programs.

Speed

Measured in MHz, determines how quickly data can be accessed. Higher speed = better performance.

Latency (CAS)

Measured in cycles, lower is better. Determines how quickly RAM responds to requests.

Dual/Triple/Quad Channel

Multiple memory channels increase bandwidth by allowing simultaneous data transfers.

How Much RAM Do You Need?

8GB

Basic computing, web browsing, office applications

16GB

Multitasking, light gaming, content creation (recommended for most users)

32GB

Heavy multitasking, serious gaming, video editing, programming

64GB+

Professional video editing, 3D rendering, virtual machines, scientific computing

Quick Facts

  • RAM is volatile memory
  • Modern RAM modules are called DIMMs
  • Laptop RAM is called SO-DIMM
  • Dual channel nearly doubles bandwidth

Choosing RAM

  • Check motherboard compatibility
  • Match CPU specifications
  • Consider your use case
  • Buy in matched pairs for dual channel

RAM Timings

Common timing format: 16-18-18-36

  • CAS Latency (CL): Most important
  • tRCD: Row to column delay
  • tRP: Row precharge time
  • tRAS: Row active time
