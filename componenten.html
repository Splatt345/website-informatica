Ontdek de essentiële bouwstenen van elke computer

Processor (CPU)

De Central Processing Unit is de hersenen van je computer. Hij voert berekeningen uit en verwerkt instructies.

Belangrijkste Specificaties:

  • Kloksnelheid (GHz)
  • Aantal cores en threads
  • Cache geheugen
  • Vermogensverbruik (TDP)

Populaire Merken:

Intel AMD

Videokaart (GPU)

De Graphics Processing Unit is verantwoordelijk voor het renderen van beelden en video's.

Belangrijkste Specificaties:

  • VRAM geheugen
  • CUDA cores / Stream processors
  • Kloksnelheid
  • Ray tracing en DLSS

Populaire Merken:

NVIDIA AMD Intel Arc
Bekijk Details

Geheugen (RAM)

Random Access Memory is het snel werkgeheugen waar je computer actieve programma's opslaat.

Belangrijkste Specificaties:

  • Capaciteit (GB)
  • Snelheid (MHz/MT/s)
  • Latency (CL)
  • Type (DDR4/DDR5)

Populaire Merken:

Corsair G.Skill Kingston
Bekijk Details

Opslag

Opslagapparaten bewaren je bestanden, besturingssysteem en programma's permanent.

Belangrijkste Specificaties:

  • Capaciteit (GB/TB)
  • Lees-/schrijfsnelheid
  • Type (SSD/HDD)
  • Interface (SATA/NVMe)

Populaire Merken:

Samsung Western Digital Crucial

Moederbord

Het moederbord is de ruggengraat die alle componenten met elkaar verbindt.

Belangrijkste Specificaties:

  • Socket type
  • Chipset
  • Form factor
  • Connectiviteit

Populaire Merken:

ASUS MSI Gigabyte

Voeding (PSU)

De Power Supply Unit levert stroom aan alle componenten in je computer.

Belangrijkste Specificaties:

  • Vermogen (Watt)
  • Efficiëntie (80+ rating)
  • Modulariteit
  • Kwaliteit en betrouwbaarheid

Populaire Merken:

Seasonic Corsair EVGA

Koeling

  • Geluidsproductie
  • Compatibiliteit
